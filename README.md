# 🌱 KrishiMitr - Farm Aid Robo Car

## 🚀 Overview
KrishiMitr is an AI-powered smart farming system using Raspberry Pi, IoT, and PyTorch.

## 🔥 Features
- Robo car control (Forward, Backward, Left, Right)
- Live camera feed
- AI-based plant disease detection (PyTorch)
- Soil moisture & temperature monitoring
- Web dashboard + Mobile app

## 🧠 Technologies Used
- Python (Flask)
- PyTorch (Deep Learning)
- Raspberry Pi 4B
- Flutter / Android
- IoT Sensors
